# 🌸 Moomin estudia

**https://moominparticipios.streamlit.app**

App de práctica de idiomas con estética Moomin. Tiene dos modos:

- **🇪🇸 Participios en español** — elige el participio correcto para completar frases con haber
- **🇷🇺 Verbos en ruso** — conjuga verbos comunes en presente (nivel A2)

En ambos modos hay pistas, feedback con GIFs animados y un cuaderno de repaso con todas las frases.
